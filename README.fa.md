یک ابزار پرمیوم، کاملاً سمت کلاینت (تحت مرورگر) جهت اسکن آی‌پی‌های تمیز و ویرایش گروهی کانفیگ‌های Xray/V2Ray. این پنل با طراحی کهکشانی پویا، تایپوگرافی مدرن، المان‌های شیشه‌ای (Glassmorphic) و حالت‌های تیره/روشن طراحی شده است.

> **۱۰۰٪ استاتیک و بدون نیاز به سرور:** پروژه هیچ نیازی به دیتابیس یا سرور پس‌زمینه ندارد و اجرای آن کاملاً داخل مرورگر کاربر صورت می‌گیرد؛ از این رو بدون نیاز به هیچ پیش‌نیازی به صورت محلی و آفلاین قابل اجراست.

---

## 🌟 ویژگی‌های کلیدی

### ۱. ⚡ اسکنر آی‌پی تحت مرورگر
اسکن مستقیم آی‌پی‌های CDN بدون نیاز به سرور واسط یا ابزار جانبی.
- **ارائه‌دهندگان CDN:** پشتیبانی از Cloudflare، Gcore، Fastly یا رنج‌های CIDR دلخواه شما.
- **اسکن پورت‌ها:** پشتیبانی از پورت‌های مهم HTTP/HTTPS شامل (443, 8443, 2053, 2083, 2096, 80, 8080).
- **عملکرد فوق‌العاده بالا:** معماری چندنخی بهینه‌سازی شده با الگوریتم تطبیق $O(1)$ که امکان بارگذاری و اسکن تا **۵۰,۰۰۰** آی‌پی کاندیدا را بدون افت سرعت یا فریز شدن مرورگر فراهم می‌کند.
- **آمارهای زنده:** نوار پیشرفت لحظه‌ای، محاسبه پینگ و نمایش نتایج مرتب‌شده بر اساس کمترین تأخیر.
- **امکانات خروجی:** کپی آی‌پی‌های سالم، دانلود لیست به صورت فایل متنی یا خروجی مستقیم به بخش کانفیگ‌ساز.

### ۲. 🛠️ تغییر و تولید گروهی کانفیگ‌ها
تغییر یا تولید صدها کانفیگ به صورت یکجا با جایگذاری آی‌پی‌های تمیز.
- **پروتکل‌های تحت پشتیبانی:** VLESS، VMESS، Trojan و WireGuard.
- **روش‌های ورودی:**
  - **رنج‌های آی‌پی (CIDR):** تولید متوالی کانفیگ‌ها از رنج‌های ورودی.
  - **لیست آی‌پی:** ترکیب یک کانفیگ پایه با لیستی از آی‌پی‌ها یا اندپوینت‌های تمیز.
  - **لیست کانفیگ:** استخراج خودکار آی‌پی‌ها از کانفیگ‌های موجود و جایگذاری در کانفیگ پایه.
  - **حالت اسپوف SNI:** تغییر و تنظیم سریع مشخصات SNI به صورت محلی.
- **خروجی سریع:** کپی یا دانلود تمامی کانفیگ‌های تولید شده به عنوان فایل متنی با یک کلیک.

### ۳. 🎨 طراحی پیشرفته رابط کاربری
- **انیمیشن پس‌زمینه کهکشان:** ذرات ستاره و سحابی‌های متحرک با شبیه‌سازی زنده روی بوم (Canvas).
- **شخصی‌سازی گسترده:** پشتیبانی از ۴ رنگ اصلی (صورتی، آبی، زمردی، بنفش) و حالت تمام‌عیار تیره/روشن.
- **ذخیره سشن به صورت آفلاین:** استفاده از سیستم کشِ محلی (Local Storage) برای ذخیره اطلاعات، تنظیمات و آخرین کانفیگ‌های شما برای استفاده‌های بعدی بدون نیاز به اینترنت.

## 🚀 نحوه اجرا و استفاده

### روش اول: اجرای مستقیم محلی (بدون نیاز به هیچ پیش‌نیازی)
1. تمامی فایل‌های پروژه را دانلود یا کلون کنید.
2. فایل **`index.html`** را پیدا کنید.
3. روی آن دو بار کلیک کنید تا در مرورگر پیش‌فرض شما (Chrome, Firefox, Safari و...) باز شود.
4. پروژه کاملاً مستقل و آفلاین روی سیستم شما اجرا خواهد شد.

### روش دوم: سرور محلی پایتون (اختیاری)
اگر می‌خواهید پنل را از طریق شبکه محلی (مثلاً برای تست روی گوشی موبایل متصل به مودم مشترک) در دسترس قرار دهید:
```bash
python3 -m http.server 8080
```
سپس آدرس [http://localhost:8080](http://localhost:8080) را در مرورگر خود باز کنید.
